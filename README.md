# Mini Amazon Shopping System

## Overview

Mini Amazon Shopping System is a simple Python terminal-based project that simulates basic features of an online shopping platform.

The program allows users to:

* View available products
* Add products to cart
* Apply coupon discounts
* Generate shopping receipts

This project was created using basic Python concepts and is suitable for beginners learning programming.


## Features

* Product inventory system
* Add to cart functionality
* Stock management
* Coupon discount system
* Receipt generation
* Menu-driven interface


## Technologies Used

* Python


## Python Concepts Used

* Dictionaries
* Lists
* Tuples
* Functions
* Loops
* Conditional statements
* String formatting
* User input handling
